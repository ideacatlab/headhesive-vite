What is it?

@headhesive-vite providing an on-demand sticky header. Optimized and updated for modern JavaScript development. View demo.

Install

Install with npm: npm install headhesive-vite

Install with yarn: yarn install headhesive-vite

How to use?

@headhesive-vite is a standalone JavaScript plugin with no dependencies. Include the headhesive.mjs (from the dist/ folder) in the footer of your page and initialise it: 

// Create a new instance of Headhesive
var header = new Headhesive('.header');

Styling @headhesive-vite

@headhesive-vite doesn't inject any CSS styles so you can use your own to completely control how your sticky element behaves. An example is provided in the demo.

Browser support

IE9+ and modern browsers.
